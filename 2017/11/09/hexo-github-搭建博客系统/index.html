Hexo + GitHub快速搭建一个博客系统

安装Hexo之前需要准备的步骤：

  1. 安装git和node（我这里使用的是6的版本）

安装hexo

  1. 使用npm命令安装

  2. 安装好hexo后，选择一个文件夹初始化好相关文件设置(我本地是新建了一个hexo文件夹)，进入文件夹后右键打开Git bash

  1. 在安装一些相应的依赖包，同样使用npm安装

  2. 安装完之后，可以通过命令启动服务，查看本地部署是否成功

  3. 在浏览器中打开本地4000端口查看本地启动结果，如果页面正常显示，表示本地安装成功

部署到GitHub上

  1. 首先需要一个GitHub账号，并创建一个仓库，记住仓库的名字需要特别注意， == 例如：你的账户名为xxxx，则仓库的名称为xxx.gtihub.io == (这里切记要一样)

  2. 然后配置初始化好的hexo文件夹，修改里面的.yml文件，将其修改为( == 最好使用sublime修改，这里需要注意的是，参数前面需要空两格 ==)：

  3. 配置好之后就可以将本地项目deploy到GitHub托管了，这里需要注意的是：==我使用的是https进行提交，你也可以配置本地的ssh进行提交，这样就可以不需要输入账号和密码了==。

  4. 成功推送到GitHub之后， == 可以直接访问你的仓库名称（xxx.github.io） == 就可以看到效果了
